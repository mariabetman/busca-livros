# Busca livros

Aplicação frontend desenvolvida com **React + TypeScript + Vite**, que permite buscar livros utilizando uma API pública gratuita.

O usuário digita o nome de um livro no input e a aplicação retorna uma lista com os resultados encontrados.

## Tecnologias utilizadas

- **Vite**
- **React**
- **TypeScript**
- **Jest**
- **Open Library API**

## Funcionalidades

- Campo de busca de livros
- Consumo de API externa (Open Library)
- Listagem de livros encontrados
- Exibição de autor e ano de publicação
- Testes automatizados com Jest
